## Typography

- **Primary Font:** Satoshi
- **Secondary Font :** Lexend or Geist
- **Font files location:** `assets/fonts/`

### Scale
- Heading 1 (Hero): `60px` / `3.75rem` — font-weight 900, line-height 1.1
- Heading 2 (Section): `36–40px` / `2.25–2.5rem` — font-weight 900
- Heading 3 (Card): `20px` / `1.25rem` — font-weight 700
- Body: `16px` / `1rem` — font-weight 400, line-height 1.6
- Small / Caption: `12–14px` / `0.75–0.875rem` — font-weight 500

## Spacing & Radius

- Base spacing unit: `4px` (Tailwind default — all spacing multiples of 4)
- Section vertical padding: `96px` / `6rem` (py-24)
- Card border radius: `24px` / `1.5rem` (rounded-3xl)
- Button border radius: `9999px` (rounded-full — pill shape)
- Input border radius: `12px` / `0.75rem` (rounded-xl)
- Icon container radius: `16px` / `1rem` (rounded-2xl)
- University logo tile radius: `16px` / `1rem` (rounded-2xl)

## Design Personality
- **Entire app:** Clean, professional, minimal. Students trust it like a serious tool.
- **Quiz environment only:** Gamified, energetic, fun. Feels like a game, works like a study tool.
- The Study Space page itself is still professional. Only the Quiz flow uses the gamified style.
- Never apply the gamified style to any other page or section.

---

## Voice & Tone
- Friendly but smart
- Encouraging, never condescending
- Speaks to students like a senior colleague who has figured things out
- Short sentences. Clear language. No jargon.
